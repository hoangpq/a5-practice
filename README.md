# Getting Started
Simple Angular app with Angular 5 and Webpack

## Development
```bash
$ npm i
$ npm run dev
```
Navigate to `http://localhost:9090/`

## Build
```bash
$ npm run build
$ cd dist
$ http-serve -p 9090
```
Navigate to `http://localhost:4200/`

This project based on `Angular 5` and `Webpack 3`
